Introduction

What is Python?

Python is a popular programming language. It was created by Guido van Rossum, and released in 1991.

It is used for

  • web development (server-side scripting)
  • software development
  • Artificial intelligence (A.I)
  • Machine Learning (M.L)

What can Python do?

  • Python can be used on a server to create web applications.
  • Python can connect to database systems. It can also read and modify files.
  • Python can be used to handle big data and perform complex mathematics.
  • Python can be used for rapid prototyping, or for production-ready software development.

Python Syntax compared to other programming languages

  • Python was designed for readability, and has some similarities to the English language with influence from mathematics.
  • Python uses new lines to complete a command, as opposed to other programming languages which often use semicolons or parentheses.
  • Python relies on indentation, using whitespace, to define scope; such as the scope of loops, functions and classes. Other programming languages often use curly-brackets for this purpose.

The most recent major version of Python is Python 3, which we shall be using in this tutorial. However, Python 2, although not being updated with anything other than security updates, is still quite popular.

Comments in Python

  • Comments can be used to explain Python code.
  • Comments can be used to make the code more readable.
  • Comments can be used to prevent execution when testing code.

Single-line Comment

Comments starts with a #, and Python will ignore them. Comments can be placed anywhere of a line, and Python will ignore the rest of the line.

comments.py

# This is an Comment
print("Demonstrating Comments in Python") # this line print text in screen

Output

Demonstrating Comments in Python

Multi-line Comment

Python does not really have a syntax for multi line comments.you can use a multiline string. Since Python will ignore string literals that are not assigned to a variable, you can add a multiline string (triple quotes) in your code, and place your comment inside it

comments.py

'''
This is an multiline comment
which will be ignored
by interpreter.
'''
print("Demonstrating Comments in Python")

As long as the string is not assigned to a variable, Python will read the code, but then ignore it, and you have made a multiline comment.

Python Variables

Creating Variables

Python has no command for declaring a variable.

A variable is created the moment you first assign a value to it.

Variables do not need to be declared with any particular type, and can even change type after they have been set.

variables.py

x = 5 # x is of type int
y = "John" # y is of type string
print(x) # prints 5
print(y) # prints "John"

# overwriting the value of x
x = "Sally" # x is now of type str
print(x) #prints "Sally"

Casting

If you want to specify the data type of a variable, this can be done with casting.

Example

x = int(3) # x will be 3
y = float(3) # y will be 3.0
z = str(3) # z will be '3'

Get the Type

You can get the data type of a variable with the type() function.

Example

x = 5
y = "John"
print(type(x))
print(type(y))

Output

<class 'int'>
<class 'str'>

Python - Variable Names

Variable Names

A variable can have a short name (like a and b) or a more descriptive name (age, name, number).
Rules for Python variables :-

  • A variable name must start with a letter or the underscore character
  • A variable name cannot start with a number
  • A variable name can only contain alpha-numeric characters and underscores (A-z, 0-9, and _ )
  • Variable names are case-sensitive (age, Age and AGE are three different variables)

Example

# Correct ways to create variables
myvar = "abc"
my_var = "abc"
_my_var = "abc"
myVar = "abc"
MYVAR = "abc"
myvar2 = "abc"

# Wring ways to create variable this will result into error
my var = "John"
my-var = "John"
2myvar = "John"

Multi Words Variable Names

Variable names with more than one word can be difficult to read.

There are several techniques you can use to make them more readable:

Camel Case

Each word, except the first, starts with a capital letter

Example

myAge = 18

Pascal Case

Each word starts with a capital letter

Example

MyAge = 18

Snake Case

Each word is separated by an underscore character

Example

my_age = 18

Python Type Casting

There may be times when you want to specify a type on to a variable. This can be done with casting. Python is an object-orientated language, and as such it uses classes to define data types, including its primitive types.

Casting in python is therefore done using constructor functions:-

  • int() - constructs an integer number from an integer literal, a float literal (by removing all decimals), or a string literal (providing the string represents a whole number)
  • float() - constructs a float number from an integer literal, a float literal or a string literal (providing the string represents a float or an integer)
  • str()constructs a string from a wide variety of data types, including strings, integer literals and float literals

Example (int type casting)

x = int(1) # x will be 1
y = int(2.8) # y will be 2
z = int("3") # z will be 3

Example (float type casting)

x = float(1) # x will be 1.0
y = float(2.8) # y will be 2.8
z = float("3") # z will be 3.0
w = float("4.2") # w will be 4.2

Example (string type casting)

x = str("s1") # x will be 's1'
y = str(2) # y will be '2'
z = str(3.0) # z will be '3.0'

Python Loops

Python has two primitive loop commands

  • While loop
  • for loop

while Loop

With the  while loop we can execute a set of statements as long as a condition is true. And there is an optional  else block normally ee use it to know whether we are out of loop or not.

Syntax

while condition:
  # code to be excuted then condition satisfied
else:
  # block of code

The   while loop requires relevant variables to be ready, in this example we need to define an indexing variable, i or anything, which we set to.

lets look for an example based on while loop.

if__else.cpp (Example)

i = 1

while i < 6:
  print(i)
  i += 1

Output

1
2
3
4
5

Note : Remember to increment the variable, or else the loop will continue forever.

for loop

A  for loop is used for iterating over a sequence (that is either a list, a tuple, a dictionary, a set, or a string).

This is less like the for keyword in other programming languages, and works more like an iterator method as found in other object-orientated programming languages.

Example

i = 1

for i in range(0,6):
  print(i)

Output

1
2
3
4
5

Note : range() is a function which take two parameters first initial value and second is last value and here all value automatically increments by 1

With the  for loop we can execute a set of statements, once for each item in a list, tuple, set etc.
