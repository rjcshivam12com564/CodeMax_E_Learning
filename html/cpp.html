Introduction

What is C++ ?

  • C++ is a cross-platform language that can be used to create high-performance applications.
  • C++ was developed by Bjarne Stroustrup, as an extension to the C language
  • C++ gives programmers a high level of control over system resources and memory.
  • The language was updated 4 major times in 2011, 2014, 2017, and 2020 to C++11, C++14, C++17, C++20.

Why to Learn C++ ?

  • C++ is one of the most popular programming language in the world.
  • C++ can be found in today's operating systems, Graphical User Interfaces, and embedded systems.
  • C++ is an object-oriented programming language which gives a clear structure to programs and allows code to be reused, lowering development costs.
  • As C++ is close to C# and Java, it makes it easy for programmers to switch to C++ or vice versa

Difference between C and C++

  • C++ was developed as an extension of C, and both languages have almost the same syntax.
  • The main diffference between C and C++ is that C++ support classes and objects, while C does not support classes

Getting started with C++

Requirements

We require only two things to become a C++ Programmer.

  • First thing is a Text Editor like notepad or an IDE like Visual Studio Code, Code Blocks, Eclipse to edit the C++ program files.
  • And the another thing is a compiler like GCC, to translate the C++ code into machine level language that the computer will understand.

First C++ Program

Leave everything behind and get ready we are going to execute our first C++ program.
Just type the below code in your text editor and run the file:-

firstprogram.cpp

#include <iostream>
using namespace std;

int main(){
    cout << "Hello World!" ;
    return 0;
}

Output

Hello World!

Don't be afraid for the code we will learn it later and if C++ is your first language then Congratulations! you have successfully run your first C++ program.

Syntax of C++

Syntax

As we already run our first C++ program let break it down and understand the following code.

firstprogram.cpp

#include <iostream>
using namespace std;

int main(){
    cout << "Hello World!" ;
    return 0;
}

1. The very first line  #include <iostream>  is a header file library that lets us work with input and output functions, such as  cout  that is used in this program.
Don't worry if you not understand then assume that this line will appear in every C program.

2. using namespace std; means that we can use names for objects and variables from the standard library.

3. The blank line is optional it just increase the readability of the code.

4. Another thing that always appear in a C++ program, is  main()  This is called a function. Any code inside its curly brackets  {}  will be executed.

5.  cout   is used to output/print text to the screen. In our example it will output "Hello World!".

6.  return 0  ends the  main()  function and results into successful termination of Program and it is optional in C++.

7.  Do not forget to add the closing curly bracket   }  to actually end the main function.

Important Points

  • Every statement in C++ ends with a semicolon  ; 
  • The compiler ignores white spaces. However, multiple lines makes the code more readable.

C++ cout statement

cout

Comments in C++

Comments can be used to explain code, and to make it more readable. It can also be used to prevent execution when testing alternative code.

Comments can be singled-lined or multi-lined.

Single-line Comment

Single-line comments start with two forward slashes  (//) .

Any text between  (//)  is ignored by the compiler (will not be executed).

Example

// This is an single line comment
cout <<"Namaste" << endl;

Multi-line Comment

Multi-line comments start with /* and ends with */.Any text between /* and */ will be ignored by the compiler.

Example

/* This is an
multi line
comment */
cout << "Namaste" << endl;

Variables in C++

Variables are containers for storing data values.
In C++, there are different types of variables (defined with different keywords), for example:

  • int - stores integers (whole numbers), without decimals, such as 123 or -123
  • float - stores floating point numbers, with decimals, such as 19.99 or -19.99
  • char - stores single characters, such as 'a' or 'B'. Char values are surrounded by single quotes
  • string - stores text, such as "Hello World". String values are surrounded by double quotes
  • bool - stores values with two states: true or false in form of 1 & 0

Declaring (Creating) Variables

To create a variable, specify the type and assign it a value

Syntax for Variable declaration

type variableName = value;

Example

int num = 123; // 123 is assign to num
float pi = 3.14; // 3.14 is assign to pi
char aToz = 'c'; // 'c' is assign to aToz
char mystr = 'Hello World'; // string 'Hello World' assign to mystr
bool flag = 1; // 1 i.e true assign to flag

Note : If you assign a new value to an existing variable, it will overwrite the previous value

C++ Variable name

All C++ variables must be identified with unique names.These unique names are called identifiers.

The general rules for naming variables are :

  • Names can contain letters, digits and underscores
  • Names must begin with a letter or an underscore ( _ )
  • Names are case sensitive (myVar and myvar are different variables)
  • Names cannot contain whitespaces or special characters like !, #, %, etc.
  • Reserved words (such as int ) cannot be used as names

Conditional statement

Conditions and If Statements

We can use conditional operators to perform different actions for different decisions.

C has the following conditional statements :-

  • Use if to specify a block of code to be executed, if a specified condition is true
  • Use else to specify a block of code to be executed, if the same condition is false
  • Use else if to specify a new condition to test, if the first condition is false
  • Use switch to specify many alternative blocks of code to be executed

The if else statement

Use the if statement to specify a block of C++ code to be executed if a condition is true.And the else statement to specify a block of code to be executed if the condition is false

Note : Else block is optional. It may be there or not.

Syntax

if (condition) {
    // block of code to be executed if the condition is true
}
else
   // block of code to be executed if the condition is false
}

if__else.cpp (Example)

int time = 20;

if (time < 18) {
   cout << "Good day." ;
}
else {
  cout << "Good evening." ;
}

Output

Good evening

Switch statement

Instead of writing many if..else statements, you can use the switch statement.

The switch statement selects one of many code blocks to be executed :-

Syntax

switch(expression) {
 case x:
  // code block
  break;

 case y:
  // code block
  break;

 default:
  // code block
}

Switch.cpp (Example)

int day = 4;

switch (day) {
 case 1:
  cout << "Monday" ;
  break;

 case 2:
  cout << "Tuesday" ;
  break;

 case 3:
  cout << "Wednesday" ;
  break;

 case 4:
  cout << "Thursday" ;
  break;

 case 5:
  cout << "Friday" ;
  break;

 case 6:
  cout << "Saturday" ;
  break;

 case 7:
  cout << "Sunday" ;
  break;
}

Output

Thursday

break keyword

When C++ reaches a break keyword, it breaks out of the switch block. This will stop the execution of more code and case testing inside the block.

When a match is found, and the job is done, it's time for a break. There is no need for more testing.

default keyword

The default keyword specifies some code to run if there is no case match

Loops in C++

Loops can execute a block of code as long as a specified condition is reached.

Loops are handy because they save time, reduce errors, and they make code more readable.

There are yotal three loops in C++ language.

  • for loop
  • while loop
  • do..while loop

for loop

When you know exactly how many times you want to loop through a block of code, use the for loop

Syntax

for(initialisation; condition; updation){
   // code block to be executed
}

forloop.cpp (Example)

for (int i = 0; i <= 5; i++){
  cout << i << endl;
}

Output

0
1
2
3
4
5

while loop

The  while loop loops through a block of code as long as a specified condition is true

Syntax

while(condition){
   // code block to be executed
}

whileloop.cpp (Example)

int i = 0;

while (i <= 5) {
  cout << i << endl;
  i++;
}

Output

0
1
2
3
4
5

Note : Do not forget to increase the variable used in the condition (i++), otherwise the loop will never end!

do..while loop

The  do/while  loop is a variant of the while loop. This loop will execute the code block once, before checking if the condition is true, then it will repeat the loop as long as the condition is true.

Syntax

do{
   // code block to be executed
} while(condition);

dowhileloop.cpp (Example)

int i = 0;

do{
  cout << i << endl;
  i++;
} while(i <= 5);

Output

0
1
2
3
4
5
